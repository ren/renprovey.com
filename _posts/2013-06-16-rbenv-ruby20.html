rbenv + Ruby 2.0.0, OS X

Is rbenv install giving you an OpenSSL error?

I peiced together the simplest solution I could find: brew install openssl export CONFIGURE_OPTS="--with-openssl-dir=`brew --prefix openssl`" rbenv install 2.0.0-p195 See: ruby-lang.org to make sure you're installing the current patch level.

A big thanks to this post [dlackty.org] for helping me avoid a mess of dated information and lib installs.
